🤟 About ISL Connect

ISL Connect is a communication platform designed to bridge the gap between deaf and hearing users.

Admin responsibilities: Manage user accounts — approve, suspend, delete, and send messages to users.

User features: Contacts, Favourites, Emergency contacts, ISL Sign Language, Speech-to-Text, Text-to-Speech, calling by User ID, ringtone selection, dark/light theme.

Version: ISL Connect v3  |  Stack: Python Flask · JavaScript Web APIs · CNN Image Classifier
