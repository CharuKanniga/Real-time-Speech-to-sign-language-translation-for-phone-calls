ℹ️ About ISL Connect

ISL Connect bridges communication between deaf and hearing users using AI-powered tools.

🎙️

Speech to Text

Hearing users speak into the mic and speech is converted to text for deaf users.

🔊

Text to Speech

Deaf users type their message and it gets spoken aloud to the other person.

🤟

Text to Sign Language

Text is converted to ISL sign images, spelled out letter by letter if needed.

📞

ID-based Calling

No phone number needed. Search by User ID, add contact, click Call.
